# 42 Common Core – Programming Fundamentals Exercises (libft)

## Overview

This repository contains a collection of exercises related to the **first major project of the 42 Common Core curriculum: libft**. The goal of these exercises is to help students practice and understand the core programming concepts required to successfully complete the project.

The repository includes implementations and example solutions for several fundamental C functions that are commonly required during the libft project.

This resource is intended for:

* 42 students preparing for the **libft project**
* Beginners learning **C programming fundamentals**
* Anyone who wants to practice standard library function re-implementations

---

## What is libft?

The **libft** project is the first core programming assignment in the 42 curriculum. Its objective is to recreate a personal version of several standard C library functions.

Through this project, students learn:

* memory management
* string manipulation
* pointer handling
* linked lists
* modular programming
* writing clean and reusable code

Mastering libft is essential before continuing with later Common Core projects.

## How to Use This Repository

You can clone the repository using:

```
git clone git@github.com:Dorad89/42_Common_Core_Exam_ProgrammingFundamentals_ExercisesWithAnswers.git
```

Then navigate into the directory:

```
cd 42_Common_Core_Exam_ProgrammingFundamentals_ExercisesWithAnswers
```

Open the files with your preferred editor (for example VS Code):

```
code .
```

Study each function implementation and try to:

* understand the logic
* rewrite the functions yourself
* compare your solution with the provided one
